## 歡迎來到 TripPuzzle！以下是您使用 TripPuzzle 的重要資訊和步驟指南。

**開始規劃**

1. 註冊 / 登入 ：第三方 or email 註冊
2. 在Google Map上查看景點
3. 立即規劃 ：
* 新增行程
* 選擇景點
* 安排景點時間
* 選擇共同編輯行程夥伴（ QR code 邀請 or 網址 ）
* 收藏喜愛景點 / 個人行程
